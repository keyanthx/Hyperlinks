Fonts bundled with this project
===============================

All fonts below are self-hosted so the page contacts no external font host
(GDPR — no visitor IP is disclosed to a third party on load).

1. Narkiss Asaf  (NarkissAsaf-Regular-TRIAL.otf, NarkissAsaf-RegularMono-TRIAL.otf)
   Designer / Foundry: Fontef Type Foundry — https://fontef.com
   License: TRIAL / EVALUATION version — NOT an open-source font.
   Used as the primary body serif and for the intro title/credit.
   NOTE: This is a trial font. Trial licenses typically forbid production and
   public web use as well as redistribution. Before keeping it on a public site,
   confirm you hold a commercial/web license from Fontef, or replace it with the
   Tinos fallback below (already configured in the CSS font stack).

2. Tinos  (tinos-400.woff2, tinos-700.woff2)
   Designer: Steve Matteson (Ascender / Monotype).
   License: SIL Open Font License, Version 1.1.
   Metric-compatible with Times New Roman; used as the fallback body serif.
   Source: Google Fonts — https://fonts.google.com/specimen/Tinos
   (Latin subset, retrieved from fonts.gstatic.com.)

3. Overpass Mono  (overpass-mono.woff2)
   Designer: Delve Withrington, Dave Bailey, Thomas Jockin (Red Hat).
   License: SIL Open Font License, Version 1.1.
   Used for captions, the footer, and the hover material-shift.
   Source: Google Fonts — https://fonts.google.com/specimen/Overpass+Mono
   (Latin subset, variable weight 300–700.)

The full text of the SIL Open Font License 1.1 (covering Tinos and Overpass Mono)
is available at: https://openfontlicense.org/
